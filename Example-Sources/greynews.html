Stay Safe & Secure

Staying safe in grey hack has never been harder, with all the zero-days an threats out there, but lets start with the basics Well, The first thing you should know is - Do Not! run binaries from places you dont trust.
Next, dont try to hack into places from your home IP, either rent a safe server or hack into npc and corrupt logs, but never risk your home for hacking(you could not hack anyone but aight).

Managing Filesystem

If you dont know, you filestructure and it's core is one of the key things to care about when wanting to keep yourself secure, and a good way to implement that would be either via coding, or via terminal, the command for permission management is chmod, if you want to find more use Manual.exe, go to
Manual.exe > Terminal Programs > Permissions and users > [Command]
there all of the permission and file management built in utilities are placed with their usage, its very important not to ignore this step since its one of the most critical parts of the game.
